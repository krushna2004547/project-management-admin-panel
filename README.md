# 🚀 Full Stack Landing Page + Admin Panel

A full-stack web application with Landing Page and Admin Panel built using Spring Boot, REST APIs, and MySQL.

## ✨ Features

* Landing Page UI
* Projects Section (Dynamic)
* Happy Clients Section
* Contact Form Submission
* Newsletter Subscription
* Admin Panel
* Project Management
* Client Management
* Contact Form Data View
* Subscribed Emails View

## 🛠 Tech Stack

Frontend:

* HTML
* CSS
* JavaScript

Backend:

* Spring Boot
* REST APIs
* JPA / Hibernate

Database:

* MySQL

## 📂 Project Structure

Controller → Service → Repository → Entity

## ▶️ Run Project

1. Clone repo
2. Configure database
3. Run Spring Boot
4. Open frontend
